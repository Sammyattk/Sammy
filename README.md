## 🌎 Sobre o Projeto

O **Agro Forte, Futuro Sustentável** é uma plataforma educacional que simula a análise de propriedades rurais do Paraná.

O objetivo é mostrar como fatores como:

- 🌱 Solo
- 🌲 Bioma
- 🌾 Culturas
- 📜 Histórico de Plantio
- 💧 Manejo Hídrico
- 🌳 Preservação Ambiental

influenciam diretamente a produtividade e a sustentabilidade agrícola.

---

## 🎯 Objetivo

Demonstrar que é possível produzir de forma eficiente sem ignorar a conservação dos recursos naturais.

O projeto foi desenvolvido com base no tema:

> **Agro Forte, Futuro Sustentável: equilíbrio entre produção e meio ambiente**

---

## 🗺️ Foco no Paraná

O sistema foi construído considerando características reais do estado.

- Nitossolo

## 🚜 Funcionalidades

### 🌱 Introdução Educativa

Apresenta informações sobre:

- Área total do Paraná

- Preservação ambiental
- Importância do agronegócio

### 🌾 Análise da Propriedade

Permite configurar:

- Região
- Solo
- Culturas

- Sistema hídrico

### 🌽 Múltiplas Culturas

Permite analisar combinações como:

- Soja + Milho
- Soja + Trigo
- Laranja + Hortaliças

---

### 📜 Histórico de Plantio

- Rotação de culturas
- Monocultura
- Impactos na sustentabilidade

### 🌲 Solos e Biomas do Paraná

## 🧠 Como o Sistema Funciona

O sistema não avalia fatores isoladamente.

- Solo e Cultura

- Produção e Conservação Ambiental

Gerando uma análise integrada da propriedade.
